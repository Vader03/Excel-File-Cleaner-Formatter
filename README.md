# Excel File Cleaner + Formatter

## Project Overview
This project automates the process of cleaning messy Excel files and exporting a formatted, analysis-ready workbook.

## Business Problem
Analysts often receive Excel files with inconsistent column names, blank rows, duplicate records, and date fields stored in different formats. This slows down reporting and increases the risk of errors.

## Tools Used
- Python
- pandas
- openpyxl
- Excel
- VS Code

## What the Script Does
- Reads an Excel file
- Standardizes column names
- Removes blank rows
- Removes duplicate records
- Converts date columns
- Exports a clean Excel file
- Applies basic formatting to the output workbook
